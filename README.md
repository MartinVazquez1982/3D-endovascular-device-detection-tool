# README

# Crear entorno en conda

Ejecutar comando

```bash
conda create --name nuevo_entorno --file requirements.txt
```

Ese comando creara el entorno e instalara todas las dependencias

# Iniciar pagina

Ejecutar comando

```bash
streamlit run app.py
```
